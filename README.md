# Dotfiles
```
My dotfiles configuration for bash, vim and others...
```
# Installation
```
git clone https://github.com/baraeerriyachy/dotfiles.git
cd dotfiles
```  
# for bash
```
cp .bashrc ~/
sudo cp bash.bashrc /etc
```

# for vim
```
cp .vimrc ~/
sudo cp .vimrc /root
``` 
# for urxvt
```
cp .Xresources $HOME
xrdb ~/.Xresources
```
