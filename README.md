# Dotfiles
```
My dotfiles configuration for bash, vim and others...
```
# Installation
```
git clone https://github.com/baraeerriyachy/dotfiles.git
cd dotfiles
```  
# for bash
```
cp .bashrc ~/
```

# for vim
```
cp .vimrc ~/
sudo cp .vimrc /root
``` 
# for urxvt
```
cp .Xresources ~/
xrdb ~/.Xresources
```
# for zsh
```
cp .zshrc ~/
```
